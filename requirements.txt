Project: Interactive Number Search
Description:
This project allows the user to input a list of numbers and then search for specific numbers within the list. The user can search for:

Exact matches
Numbers greater than a given number
Numbers less than a given number
Numbers within a specific range
Features:
Input list of numbers: The user can input multiple numbers to create a list.
Menu options for search:
Search for an exact number
Search for numbers greater than a given number
Search for numbers less than a given number
Search for numbers in a range
Display search results: The program will display whether the search was successful and show the matching numbers.
Option to continue or exit: After each search, the user can perform another search or exit the program.
